# P06 — Taskfile

Replace ad-hoc `helm template` and `helm install` commands used during T06–T17 with a `Taskfile.yaml` at repo root. Used locally and called by CI (P07).

## Sub-items

- [x] Add `Taskfile.yaml` with the following tasks:

  **`lint`**
  ```
  helm dependency update charts/sympozium-events/
  helm lint charts/sympozium-events/ -f charts/sympozium-events/ci/two-routes.yaml --strict
  ```

  **`template`**
  Renders and diffs the chart against `ci/two-routes.yaml`. Useful during template authoring.
  ```
  helm template sympozium-events charts/sympozium-events/ -f charts/sympozium-events/ci/two-routes.yaml
  ```

  **`test`**
  Runs helm unittest suite (requires `helm plugin install https://github.com/helm-unittest/helm-unittest`).
  ```
  helm unittest charts/sympozium-events/
  ```

  **`test:lint`**
  Shortcut: lint + test in sequence.

  **`package`**
  Packages the chart to `dist/`:
  ```
  mkdir -p dist
  helm package charts/sympozium-events/ -d dist/
  ```

  **`release`**
  Intended for CI use only (P07). Documents the chart-releaser invocation for reference.

  **`deps`**
  Installs local tooling prerequisites: `helm`, `helm-unittest` plugin, `task` (Go Task), `chart-releaser`. Checks if each is present before installing.
