# P07 — GitHub Actions: CI + chart-releaser

Two workflows. P05 (test suite) and P06 (Taskfile) must be complete before this task starts.

## Sub-items

### Workflow 1: `ci.yaml` (runs on PR + push to main)

- [x] Create `.github/workflows/ci.yaml`:
  - Trigger: `pull_request`, `push` (branches: `main`)
  - Steps:
    1. Checkout
    2. Set up Helm (pin version matching `.tool-versions`)
    3. Install `helm-unittest` plugin
    4. `helm dependency update charts/sympozium-events/`
    5. `task lint` — helm lint with `--strict`
    6. `task test` — helm unittest
    7. `task template` — `helm template` with `ci/two-routes.yaml`; diff output to catch regressions
  - Cache: `~/.cache/helm` between runs

### Workflow 2: `release.yaml` (runs on tag push `v*`)

- [x] Create `.github/workflows/release.yaml`:
  - Trigger: `push` (tags: `v*`)
  - Steps:
    1. Checkout (fetch full history for chart-releaser)
    2. Configure Git identity (chart-releaser commits to `gh-pages`)
    3. Set up Helm
    4. `helm dependency update charts/sympozium-events/`
    5. Run `chart-releaser` action (`helm/chart-releaser-action@v1`):
       - `charts_dir: charts` (chart is at `charts/sympozium-events/`)
       - Publishes packaged chart to GitHub Releases
       - Updates `index.yaml` on `gh-pages` branch
  - Requires: `CR_TOKEN` secret (GitHub token with `contents: write`)
