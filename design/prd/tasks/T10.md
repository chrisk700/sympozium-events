# T10 — Default taskTemplates

Per-source-type default `taskTemplate` values, overridable in `routes[].targets[].taskTemplate`.

## Sub-items

- [x] PagerDuty webhook default: incident title, service name, urgency, URL
- [x] GitHub webhook default: PR title, author, branch (for `pull_request` events)
- [x] Generic webhook default: raw body summary prompt
- [x] Wire defaults into `values.yaml` under `defaults.taskTemplates.<source-type>`; Sensor template uses the default when a target omits `taskTemplate`
